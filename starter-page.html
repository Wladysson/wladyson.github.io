Visão Geral

Este repositório documenta laboratórios reais executados durante minha formação em Computação em Nuvem pela AWS através do programa AWS Restart. Cada diretório contém um exercício prático com:

  • Descrição técnica e pedagógica
  • Capturas de tela reais
  • Reflexões e boas práticas
  • Comandos, validações e resultados

Este material representa não só a execução técnica, mas também meu comprometimento com aprendizado contínuo e excelência profissional focado totalmente à Computação e Infraestrutura na Nuvem AWS (Amazon Web Service).

Breve descrição dos níveis

  • Básico: Fundamentos de cloud, provisionamento manual e automação simples. Ideal para quem está começando.
  • Intermediário: Integração de containers, orquestração, pipelines de CI/CD e automação de infraestrutura mais robusta.
  • Avançado: Projetos focados em segurança, alta disponibilidade, observabilidade, serverless e modularização avançada.

Como começar

  • Escolha o laboratório de acordo com seu nível atual.
  • Leia o README dentro de cada pasta para instruções detalhadas.
  • Execute os scripts e pratique os comandos fornecidos.
  • Contribua adicionando melhorias e novos labs!
